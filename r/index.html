R Project

Since 2004, I have contributed code, bug fixes and documentation to the R Project. The most visible changes for the user include added functionality in aggregate, barplot, boxplot, and type.convert.

CRAN packages

I am the first author of nine packages on CRAN,

  • areaplot - Stacked areas and confidence bands
  • gmt - Interface between the GMT map-making software and R
  • icesAdvice - Functions related to ICES advice
  • icesTAF / TAF - Transparent Assessment Framework support functions
  • MSSQL - Explore MS SQL Server databases
  • ora - Explore Oracle databases and import data into R
  • plotMCMC - MCMC diagnostic plots
  • r2d2 - Two-dimensional regions and distributions
  • scape - Statistical catch-at-age plotting environment

and have contributed functions to other packages on CRAN:

  • coda - cumuplot
  • DescTools - PlotArea
  • gdata - Args, env, is.what, keep, ll
  • geo - geoSR
  • glmmTMB - ranef
  • gplots - rich.colors
  • gtools - ASCIIfy
  • icesDatras - parseDatras (coauthor)
  • icesSAG - findAssessmentKey, getSAG (coauthor)
  • icesVocab - findAphia
  • xtable - autoformat, xalign, xdigits, xdisplay
