TMB user environment

Template Model Builder (TMB) is a statistical software package written by Kasper Kristensen, at the Technical University of Denmark.

This webpage presents user environments for TMB, where users can choose from three related options:

1  Emacs tmb-mode

“C-c C-c ftw”

Lightweight Emacs mode to develop TMB models efficiently. It follows standard conventions and does not modify Emacs core keybindings. Designed for users who already use Emacs as their main editor.

Download (21 KB) from the TMB repository:

  • tmb.el

2  AD Studio

“Statistical modelling in ADMB and TMB”

Development environment that makes the convenient features of Emacs tmb-mode available to non-Emacs users. Similar to option #1, but with keybindings that mimic basic editors. Designed for users who would like a working environment that is specialized for developing TMB models, without learning Emacs at the same time.

Download (280 MB) from the AD Studio repository.

Comes with a user manual.

3  Virtual TMB

“Template Model Builder in a box”

Virtual machine optimized for developing TMB models. Requires considerable disk space (one 6 GB file), but this virtual machine is a clean solution that can be removed without a trace. It builds and runs TMB models much faster than on the Windows side, and provides better debugging facilities. Designed for Windows and Mac users—who will incidentally also become Linux users.

Download (6 GB) from the Hafro server:

  • VirtualTMB.vdi

Comes with a user manual.
