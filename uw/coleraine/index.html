Welcome
This scratchpad is where I distribute various files among colleagues and share my current thoughts about the Coleraine fisheries stock assessment software. Visit the official Coleraine website for details about the software and theory.

Utilities
[Excel] PhasePaint is a Visual Basic subroutine that colours prior parameters on the input sheet according to estimation phase.

[R] scape is an R package that provides a suite of multipanel plotting functions for Coleraine output.
